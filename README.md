# Express Lab

A lab for learning how to create a JSON API with Express.

## Installation

* [Install Node](http://nodejs.org/)

```
# Install dependencies
$ npm install
...
```

## Running

```
# Verify that everything is working
$ npm test

# Start a test watcher
$ npm run test-watch

# Start a server
$ npm start

# Start a server with file watching
$ npm run watch
```
